Hướng dẫn sử dụng:
Ứng dụng được viết trên nền WinUI 3. Để sử dụng cần cài đặt Visual Studio (2022+) đã được cài các phần bổ trợ liên quan đến WinUI 3.
![image](https://github.com/user-attachments/assets/a61a1419-2ac2-40ab-b465-fafa96ed0fcc)

Sau khi khởi động ứng dụng, click vào nút Settings góc trái trên cùng để kiểm tra các cài đặt.

![image](https://github.com/user-attachments/assets/6f70e960-bcd1-4326-a702-a58fe44055de)


Nếu các cài đặt đã đúng, bấm vào nút Start 3 lần để chạy kiểm thử.
Để bắt đầu quay số, bấm Enter. Để dừng lại ở từng cột, bấm Enter. Bấm Enter để tiếp tục quay số.

![image](https://github.com/user-attachments/assets/4954c6e3-7d92-447d-90ef-34d9f78cd589)


Để thay đổi background, vào Assets, thay đổi background.png thành ảnh của bạn.
